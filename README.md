# 项目简介

## 核心功能
- 基于YOLOv8的轮胎缺陷检测系统，支持识别4类缺陷：Bulge、Crack、Fine Crack、Nail
- 包含3864张标注数据集和训练好的模型
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 包含完整源码、UI界面源文件、项目运行教程和所需环境配置

## 适用场景
- 轮胎制造质量检测
- 汽车维修厂轮胎检查
- 轮胎回收评估
- 工业自动化生产线轮胎质检
- 研究机构轮胎缺陷分析

## 技术特点
- 基于Python 3.9开发，使用PyCharm或VSCode作为开发环境
- 依赖OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 采用YOLOv8模型进行目标检测
- 提供图形用户界面，操作简便
- 已完成模型训练，可直接部署使用
